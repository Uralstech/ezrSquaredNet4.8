# ezr² Net4.8

## What is ezr² Net4.8?
**This is not the main repo of ezr².** This is a clone of ezr² made with `DotNet 4.8` and `C# 9.0` as a `class library`,
to be used in other programs. Check out ezr² [here](https://github.com/Uralstech/ezrSquared).
